# geo-app

A web app that visualizes earthquakes that occurred in the Philippines filtered by dates.

## Getting Started

### Clone the repo   


```
git clone https://github.com/iinosantos/geo-app20180423.git
cd geo-app20180423
```

### Install packages
npm
```
npm install
```
yarn
```
yarn
```

### Run the demo
npm
```
npm run start
```
yarn
```
yarn start
```
